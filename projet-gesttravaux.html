GestTravaux Pro — Symfony

Application web multi-acteurs pour la gestion de travaux immobiliers (IMMOSYNC)

Symfony 7 Doctrine ORM MySQL PWA Twig

Présentation du Projet

GestTravaux Pro est un système de gestion de travaux immobiliers développé pour la société IMMOSYNC. L'application Symfony 7 constitue la face web du projet : elle centralise les accès des inspecteurs terrain (PWA mobile), des entrepreneurs, des propriétaires et des administrateurs.

Elle partage une base de données MySQL unique avec l'application Java Desktop, garantissant la cohérence des données entre tous les acteurs.

Acteurs & Interfaces

Inspecteur (PWA Mobile)

Consultation des chantiers assignés, localisation sur carte, saisie d'observations, prise de photos, ajout de documents (DPE, diagnostic bruit…)

Espace Entrepreneur

Dashboard des demandes de chantiers, consultation du devis type, soumission de propositions avec formulaires validés par Symfony, suivi des statuts

Espace Propriétaire

Consultation des chantiers liés à ses biens, validation des devis préliminaires, suivi de l'avancement des travaux

Administration Système

Gestion des utilisateurs, des rôles, des paramètres applicatifs et supervision globale depuis une interface web sécurisée

Fonctionnalités Clés

Carte Interactive

Localisation des biens immobiliers sur une carte pour les inspecteurs lors de leurs visites terrain

Gestion des Devis

Consultation du devis type, soumission des propositions par les entrepreneurs avec leurs prestations détaillées

Documents & Photos

Upload et gestion des fichiers d'inspection (DPE, diagnostics, photos) directement depuis le terrain

Notifications Email

Alertes automatiques aux entrepreneurs lors de nouvelles propositions de chantiers

Technologies Utilisées

Backend

  • Symfony 7 - Framework PHP
  • Doctrine ORM - Mapping objet-relationnel
  • Symfony Security - Authentification & rôles
  • Symfony Forms - Validation des formulaires

Frontend

  • Twig - Moteur de templates
  • Bootstrap 5 - Framework CSS responsive
  • JavaScript ES6+ - Interactions dynamiques
  • PWA - Progressive Web App (offline)

Base de données & Outils

  • MySQL 8 - BDD partagée avec Java
  • Git - Gestion de versions
  • PhpStorm - IDE
  • Docker - Environnement de dev

Défis Techniques

Architecture partagée

Conception d'un schéma de base de données MySQL compatible à la fois avec Doctrine ORM (Symfony) et Hibernate (Java), accessible simultanément par les deux applications.

PWA pour inspecteurs terrain

Mise en place d'une Progressive Web App permettant la saisie et l'upload de documents depuis smartphone ou tablette, avec une expérience fluide en mobilité.

Sécurité multi-rôles

Gestion fine des accès avec Symfony Security : chaque acteur (inspecteur, entrepreneur, propriétaire, admin) dispose de permissions différenciées sur les mêmes ressources.
