GestTravaux Pro

Système de gestion de travaux immobiliers (Gestion de syndic copropriété)

Symfony 7 JavaFX MySQL PWA

Présentation du Projet

GestTravaux Pro est une solution complète de gestion de travaux immobiliers développée avec Symfony 7 pour la partie web et JavaFX pour l'application desktop. Ce projet a été conçu pour faciliter la gestion des chantiers, des devis et des interventions dans le secteur du bâtiment.

L'application offre une double interface : une version web progressive (PWA) accessible depuis n'importe quel navigateur, et une application desktop Java pour une utilisation offline complète.

Fonctionnalités Principales

Gestion des Chantiers

Création, suivi et clôture des chantiers avec gestion des échéances et des ressources

Système de Devis

Génération automatique de devis détaillés avec calcul des coûts et des marges

Comparateur Multi-critères

Comparaison des fournisseurs et des matériaux selon plusieurs critères (prix, qualité, délai)

Tableaux de Bord

Visualisation en temps réel des indicateurs clés de performance (KPI)

Gestion des Équipes

Attribution des tâches, suivi des interventions et planning des équipes

Interface Responsive

Application web adaptée à tous les écrans (ordinateur, tablette, smartphone)

Technologies Utilisées

Backend & Framework

  • Symfony 7 - Framework PHP moderne
  • Doctrine ORM - Gestion de la base de données
  • API Platform - Création d'API REST
  • Symfony Security - Authentification et autorisation

Frontend Web

  • Twig - Moteur de templates
  • Bootstrap 5 - Framework CSS
  • JavaScript ES6+ - Interactions dynamiques
  • PWA - Progressive Web App

Application Desktop

  • JavaFX - Interface graphique
  • FXML - Design des interfaces
  • Scene Builder - Outil de design visuel

Base de Données & Outils

  • MySQL 8 - Base de données relationnelle
  • Docker - Conteneurisation
  • Git - Gestion de versions
  • PhpStorm - IDE principal

Défis Techniques Relevés

Architecture Hybride

Conception d'une architecture permettant de partager les données entre l'application web et desktop via une API REST sécurisée.

Performance

Optimisation des requêtes SQL avec Doctrine pour gérer efficacement de grandes quantités de données (chantiers, devis, factures).

Sécurité

Mise en place d'un système d'authentification robuste avec gestion des rôles et permissions (admin, chef de chantier, ouvrier).
