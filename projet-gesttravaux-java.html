GestTravaux Pro — Java

Application desktop pour la gestion globale des chantiers immobiliers (IMMOSYNC)

Java JavaFX Hibernate ORM MySQL Géolocalisation

Présentation du Projet

GestTravaux Pro est un système de gestion de travaux immobiliers développé pour la société IMMOSYNC. L'application Java Desktop est l'outil principal du gestionnaire : elle centralise le pilotage global des chantiers, la gestion des entités métier, la sollicitation des entrepreneurs et la comparaison des devis.

Elle accède directement à la même base de données MySQL que l'application Symfony, garantissant la cohérence en temps réel entre les deux interfaces.

Fonctionnalités Principales

Gestion des Chantiers

Création de dossiers chantier, définition du devis type avec prestations, assignation des inspecteurs et consultation des rapports d'inspection

Gestion des Entités

Administration complète : entrepreneurs, catégories d'entrepreneurs, propriétaires, biens immobiliers, prestations et catégories de prestations

Sollicitation Géographique

Algorithmes de calcul de distances pour sélectionner et contacter automatiquement les entrepreneurs qualifiés dans un rayon défini autour du bien

Comparateur de Devis

Outil d'aide à la décision multi-critères : comparaison des propositions reçues sur le prix et la date de début des travaux, avec interface graphique enrichie

Tableaux de Bord

Visualisation des indicateurs clés de suivi des chantiers, budgets et délais pour le gestionnaire

Génération de Rapports

Export PDF des devis, comparatifs et bilans de chantier pour archivage et communication

Défis Techniques

Comparateur multi-critères

Conception d'un algorithme de comparaison pondérée des devis sur deux critères : la date de début des travaux la plus proche et le prix le moins élevé, avec une interface graphique lisible et contrastée.

Calcul géographique

Implémentation d'un algorithme de calcul de distances (formule de Haversine) pour filtrer automatiquement les entrepreneurs dans un rayon géographique autour du bien à rénover.

Base de données partagée

Coordination avec l'application Symfony pour partager un schéma MySQL commun via Hibernate, tout en assurant la compatibilité avec les entités Doctrine de la partie web.
